Readme

-Unzip the mod to the Stardew Valley "Mods" folder
-Launch the game and exit it
-Go to the mod folder and open the config.json file
-Replace "Numpad1" with one of these key bindings : https://stardewvalleywiki.com/Modding:Player_Guide/Key_Bindings (for example, "D0")
-Save the config.json file and launch the game
